nvm to manage node version

NVM is Node Version Manager is a tool to manage multiple node.js version. list installed version ❯ nvm ls v8.17.0 -> v12.9.0 system default -> 12.9 (-> v12.9.0) node -> stable (-> v12.9.0) (default) stable -> 12.9 (-> v12.9.0) (default) list available version on remote (is version 14 available? If so it will list) ❯ nvm ls-remote 14 v14.0.0 v14.1.0 ... v14.13.1 v14.14.0 Install the version that you would like ❯ nvm install 14 Set version 14 (or another version) as default nvm alias default 14 Switch to version 8 (or another version) ❯ nvm use v8 Now using node v8.17.0 (npm v6.14.8) Cheers!
